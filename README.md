#商品推荐系统
结果展示
======
此工程用来展示《商品推荐系统》的分析结果。
采用JQuery技术，使用D3.js进行表格的绘制。由于采用csv格式保存数据，在D3中，若要使用相应的csv转化功能，需要搭建http服务器，搭建服务器比较简单的方法便是采用python：
```python
python -m http.server #python3.x
```
